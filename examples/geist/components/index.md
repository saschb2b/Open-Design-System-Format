# Components

Reusable UI elements — Geist's full public component library. Each ships a self-rendering
`*.example.html` (styled by [`/styles/components.css`](/styles/components.css)) that re-themes
light↔dark via the functional tokens. Geist ships as React components; these capture the tokens and
reference structure.

* [Copy button](copy-button.md) - Copies a value and flips to a check. ([example](copy-button.example.html))
