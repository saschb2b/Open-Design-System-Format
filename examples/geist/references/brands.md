The [brands page](https://vercel.com/geist/brands) documents how to work with **brand assets** — both
Vercel's own mark and colors and a large palette of **technology and partner brand colors** used across
the Vercel ecosystem (Next.js, React, frameworks, integrations). Vercel's brand is pure monochrome
(`#000` / `#fff`) — the same high-contrast stance as [Geist's color system](/foundations/color.md);
partner colors are each brand's official value, used only for that brand's logo/identity, never as UI
accents.

# Key points
- **Vercel brand:** black `#000000` and white `#ffffff`; the triangle mark. Keep it monochrome.
- **Partner colors** (representative): Next.js `#000`, React `#149eca`, Turborepo `#ff1e56`,
  TypeScript `#3178c6`, Tailwind `#06b6d4`, Vue `#42b883`, Svelte `#ff3e00`, Node `#539e43`. The full
  grid lives at the source.
- Use a brand color **only** for that brand's logo, never to theme your own UI — UI color comes from
  the [Geist scales](/foundations/color.md).

# Do & Don't
- **Do** use official brand values for logos and keep Vercel's own brand monochrome.
- **Don't** repurpose partner brand colors as accents in your interface.
