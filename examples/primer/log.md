# Primer design-system update log

## 2026-06-23
* **Update**: Expanded the component set from 22 to **45** — added button-group, textarea, autocomplete, token, select-panel, action-menu, dialog, popover, nav-list, tree-view, tab-nav, header, footer, page-layout, page-header, data-table, timeline, details, truncate, branch-name, avatar-stack, circle-badge, and skeleton (each with a runnable example), plus the supporting CSS. Comprehensive coverage of Primer's documented component surface.
* **Creation**: Initial Primer (GitHub) bundle for ODSF v0.1. Tokens read directly from the live primer.style DOM (`@primer/primitives`) in both light and dark modes — the bundle ships Primer's own variable names (`--fgColor-default`, `--button-primary-bgColor-rest`, …) so generated code matches real Primer. Includes foundations (color, typography, spacing, elevation, shape, motion, layout), 22 components with runnable examples across actions (button — shown light+dark, icon-button, link), forms (text-input, select, checkbox, radio, toggle-switch, segmented-control), status & feedback (label, state-label, banner, spinner, progress-bar, tooltip), navigation (underline-nav, breadcrumbs, pagination, action-list), and content (box, avatar, blankslate); the repository-page pattern, color-modes and focus behaviors, two guidelines with counter-examples (use-functional-tokens, color-not-alone), two references, and `styles/tokens.css` (light + dark + auto) plus `styles/components.css`.
