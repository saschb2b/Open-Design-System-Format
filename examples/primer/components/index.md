# Components

Reusable UI elements. Each ships a self-rendering `*.example.html` an agent can copy from, styled by
[`/styles/components.css`](/styles/components.css) and re-themeable via the functional tokens.

* [Button](button.md) - Default, the green primary, danger, invisible. ([example](button.example.html))
* [Button group](button-group.md) - Related buttons joined into one unit. ([example](button-group.example.html))
* [Icon button](icon-button.md) - Square, icon-only action for toolbars. ([example](icon-button.example.html))
* [Link](link.md) - The accent inline text link, plus a muted variant. ([example](link.example.html))
